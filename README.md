# AYNX

My personal portfolio website, built from scratch with Next.js and TypeScript.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Email:** Resend
- **Deployment:** Vercel

## Features

- Server-side rendering
- Contact form with rate limiting
- Responsive design
- Custom typography (Inter + Syncopate)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

```
RESEND_API_KEY=your_resend_api_key
```
